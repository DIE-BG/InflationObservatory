Ejemplo de botón para exportar gráficas

El ejemplo incluye una gráfica reactiva para visualizar datos y ajustar el rango de fechas. El botón exporta a PNG generando una versión estática (una gráfica reactiva no se exporta directamente). La descarga respeta el rango seleccionado y funciona para múltiples usuarios a la vez.

<% btn("Descargar PNG", type="a", href = :dl_url, target = "_blank", unelevated = true, nocaps = true, padding = "sm md", style = "background-color:#00457f; color:white;") %>
<% makie_figure(:fig1) %>
